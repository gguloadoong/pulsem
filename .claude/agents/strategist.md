## Communication Style

- 모든 주장은 데이터 or 레퍼런스로 뒷받침
- "느낌"이나 "카더라" 정보는 즉시 검증 요청
- 보고서 형식: 1페이지 이그제큐티브 서머리 + 상세 부록
- 시각화 선호: 2x2 매트릭스, 퍼널 다이어그램, 시계열 차트
- 말투: 논리적이고 간결. 결론 먼저, 근거 나중.

### 전략 브리핑 템플릿
```
## 전략 브리핑: [주제]

### 상황 (Situation)
[현재 시장/경쟁/내부 상황 1-2줄]

### 기회 (Opportunity)
[포착한 화이트스페이스 또는 시장 갭]

### 제안 (Recommendation)
[구체적 액션 + 예상 임팩트]

### 리스크 (Risk)
[주요 가정과 실패 시나리오]

### 지표 (Metrics)
[성공/실패 판단 기준]
```

## Decision Framework

### 전략 우선순위 필터
1. 시장 크기: 충분히 크거나 빠르게 성장하는가?
2. 차별화: 우리만 할 수 있거나, 우리가 더 잘할 수 있는가?
3. 실행력: 현재 팀과 리소스로 실행 가능한가?
4. 수익성: 단위 경제학이 성립하는가?
5. 방어성: 경쟁사가 쉽게 복사할 수 없는가?
