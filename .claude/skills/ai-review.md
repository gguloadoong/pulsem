# AI 리뷰 스킬

## 목적
AI(Claude, Copilot, ChatGPT 등)가 생성한 코드, 콘텐츠, 문서를 사람이 직접 검토하여 품질, 보안, 정확성, 팀 컨벤션 준수를 확인한다. AI 생성물은 검토 없이 프로덕션에 반영하지 않는다.

## 트리거 조건
- AI 생성 코드를 PR에 포함하기 전
- AI가 작성한 콘텐츠(뉴스 요약, 투자 인사이트)를 서비스에 반영하기 전
- AI가 작성한 문서(PRD, 테스트 케이스, API 스펙)를 확정하기 전
- AI 추천 아키텍처/기술 결정을 채택하기 전

---

## 리뷰 유형별 체크리스트

### A. AI 생성 코드 리뷰

**보안 체크 (필수):**
- [ ] API 키, 시크릿, 하드코딩된 자격증명 없음
- [ ] SQL 인젝션 취약점 없음 (파라미터 바인딩 사용)
- [ ] XSS 취약점 없음 (사용자 입력 이스케이프)
- [ ] 인증/인가 로직 정확함
- [ ] 에러 메시지에 내부 정보 노출 없음

**코드 품질 체크:**
- [ ] TypeScript 타입 정확함 (any 사용 없음)
- [ ] 팀 코딩 컨벤션 준수 (네이밍, 파일 구조)
- [ ] 불필요한 복잡도 없음 (과도한 추상화)
- [ ] 에러 핸들링 적절함 (빈 catch 없음)
- [ ] 성능 안티패턴 없음 (N+1 쿼리, 불필요한 리렌더링)
- [ ] 주석 필요한 복잡 로직에 주석 있음

**기능 정확성 체크:**
- [ ] 수용 기준의 요구사항 정확히 구현
- [ ] 엣지케이스 처리됨 (빈 배열, null, 0, 최대값)
- [ ] 실제로 동작하는 코드인가 (AI 환각 패턴 주의)

**AI 환각 주의 패턴:**
```
위험 신호:
- 존재하지 않는 라이브러리 import
- 존재하지 않는 API 엔드포인트 호출
- 실제와 다른 함수 시그니처
- 설치되지 않은 npm 패키지 사용
- 실제 동작하지 않는 코드 (컴파일은 되지만 런타임 에러)
```

### B. AI 생성 콘텐츠 리뷰 (투자 정보)

**정확성 체크 (투자 콘텐츠 특히 중요):**
- [ ] 수치 데이터가 실제 출처와 일치하는가?
- [ ] 회사명, 티커, 날짜 오류 없음
- [ ] 과거 사실을 현재 사실로 오해할 소지 없음
- [ ] 투자 권유로 해석될 수 있는 표현 없음

**규제 컴플라이언스:**
- [ ] "매수/매도 추천" 표현 없음
- [ ] "확실히 오른다/내린다" 단정적 표현 없음
- [ ] 정보 출처 명시 또는 "AI 생성 요약" 표시
- [ ] 투자 판단은 개인 책임임을 명시

**콘텐츠 품질:**
- [ ] 문법, 맞춤법 오류 없음
- [ ] PulseM 톤앤매너 준수
- [ ] MZ 친화적이되 전문성 유지
- [ ] 핵심 정보가 빠지지 않음

### C. AI 생성 문서 리뷰 (PRD, 테스트케이스 등)

- [ ] 실제 프로젝트 맥락에 맞는 내용인가?
- [ ] 팀 컨벤션과 용어가 일치하는가?
- [ ] 일반적인 내용과 PulseM 특화 내용이 구분되는가?
- [ ] 실행 가능하고 측정 가능한 내용인가?

## 실행 순서

### Step 1: AI 생성물 식별
- AI 생성 코드/콘텐츠임을 명확히 표시
- 어떤 AI 도구, 어떤 프롬프트로 생성했는지 기록 (가능하면)

### Step 2: 체크리스트 기반 리뷰
- 유형에 맞는 체크리스트 실행
- 각 항목을 실제로 확인 (체크만 하는 것 금지)
- 의심스러운 부분은 실제 실행/테스트로 검증

### Step 3: 문제 분류 및 수정
```
즉시 수정 필요 (반영 불가):
  - 보안 취약점
  - 사실 오류 (투자 콘텐츠)
  - 규제 위반 표현
  - 컴파일/런타임 에러

수정 권장 (반영 가능하나 개선 필요):
  - 코딩 컨벤션 불일치
  - 과도한 복잡도
  - 불명확한 표현

수용 가능 (주석만 추가):
  - 스타일 차이 (팀 컨벤션 범위 내)
  - 대안이 있으나 현재도 무방한 패턴
```

### Step 4: 리뷰 기록
- 리뷰 결과 PR 코멘트 또는 Notion에 기록
- AI 생성물임을 코드 주석 또는 문서에 표시

## 출력 형식

```markdown
# AI 리뷰 결과

**대상:** [PR 번호 / 문서명 / 콘텐츠 제목]
**AI 도구:** Claude / Copilot / ChatGPT / 기타
**리뷰어:** [이름]
**리뷰 일시:** YYYY-MM-DD

## 체크리스트 결과
보안: ✅ 통과 / ❌ 문제 발견
코드 품질: ✅ / ⚠️ 개선 필요 / ❌
기능 정확성: ✅ / ❌
규제 컴플라이언스: ✅ / N/A

## 발견된 문제
### 즉시 수정 필요
- [문제 설명]: [위치] → [수정 방향]

### 수정 권장
- [문제 설명]: [위치] → [수정 방향]

## 최종 판정
**반영 가능 ✅ / 수정 후 반영 ⚠️ / 반영 불가 ❌**

## 주석 추가
코드/문서에 아래 표시 추가:
`// AI-generated: Claude (검토: [이름], YYYY-MM-DD)`
```

---

## 규칙
- AI 생성 코드는 반드시 사람이 리뷰 후 머지
- 투자 관련 AI 콘텐츠는 반드시 사실 검증 후 게시
- AI 생성물에 "AI 생성" 출처 표시 (서비스 내 콘텐츠 포함)
- 보안 체크리스트 항목은 생략 없이 전체 실행
- AI가 AI 생성물을 리뷰하는 것으로 완전 대체 불가 (최종 확인은 사람)
