# Skill: loop — 자율 팀 운영 사이클 (오토파일럿)

**트리거:** kickoff 이후 자동 또는 "loop 스킬 실행해"
**실행 흐름 (무한 루프):**
- Step 1: backlog.md, decisions.md, 최근 코드 파악
- Step 2: PM/Strategist가 P1/P2 검토 + 새 기회 발굴
- Step 3: 지정 에이전트가 코드/문서/테스트 수행
- Step 4: 발견 이슈 → backlog.md + GitHub Issue 등록
- Step 5: 반복 문제 발견 시 team-meeting 호출
- Step 6: 태스크 남으면 Step 1 복귀. 백로그 비어도 자발적 탐색 지속.
*종료:* CEO 개입 필요 시만 일시 정지.
