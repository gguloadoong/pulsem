# PulseM — 제품 요구사항 문서 (PRD)

**버전:** 1.0.0
**최종 수정:** 2026-03-21
**작성자:** PulseM 팀

---

## 1. 제품 개요

### 1.1 배경 및 문제 정의

국내 개인 투자자 수는 2020년 이후 폭발적으로 증가하여 2024년 기준 주식 계좌 보유자만 1,400만 명을 넘어섰다. 하지만 대부분의 투자자들은 다음과 같은 고통을 겪고 있다.

**핵심 페인 포인트:**
1. **정보 파편화** — 주식은 증권사 앱, 코인은 업비트/빗썸, 부동산은 네이버 부동산·호갱노노를 각각 확인해야 함. 하루에 최소 4~5개 앱을 오가며 투자 현황을 파악
2. **규제/뉴스 누락** — 금융위원회 공시, 기준금리 변경, 암호화폐 규제안 같은 시장 교란 요인을 뒤늦게 인지하여 대응 타이밍을 놓침
3. **습관 형성 실패** — 투자 공부를 하려 해도 지속 동기가 없음. 앱을 열어도 단순 숫자 나열에 그쳐 학습 동기 부여가 안 됨
4. **커뮤니티 신뢰성 부재** — 기존 투자 커뮤니티는 검증되지 않은 정보와 종목 추천이 범람하여 신뢰성이 낮음

### 1.2 솔루션

PulseM은 **게이미피케이션 메커니즘**을 핵심 엔진으로 삼아, 투자 정보 소비를 습관으로 만드는 올인원 투자 정보 플랫폼이다.

- 주식·코인·부동산 정보를 한 곳에서 실시간으로 확인
- 중요 규제·뉴스 알림을 놓치지 않도록 스마트 알림 제공
- 일일 퀘스트, XP, 뱃지, 랭킹을 통해 투자 습관 형성 지원
- 검증된 데이터 기반의 인사이트 제공

### 1.3 비전

> "투자의 맥박을 실시간으로 느끼는 게이미피케이션 투자 정보 플랫폼"

3년 안에 국내 20-40대 직장인 투자자들이 매일 아침 출근길에 가장 먼저 여는 투자 앱이 된다.

## 2. 타겟 사용자

### 2.1 주요 페르소나

#### 페르소나 A: 김준호 (32세, 대기업 IT 직군)
- **직업:** 서울 소재 IT 기업 개발자, 연봉 6,500만 원
- **투자 경력:** 3년차 (주식 2년, 코인 1년)
- **투자 규모:** 주식 4,000만 원, 코인 500만 원
- **행동 패턴:**
  - 출근 전 30분, 점심시간 10분, 퇴근 후 20분 투자 정보 확인
  - 여러 앱을 전환하며 시간을 낭비한다고 느낌
  - 테크 친화적이라 새로운 앱 수용성 높음
- **페인 포인트:** "주식 앱, 코인 앱, 뉴스 앱을 왔다 갔다 하는 게 너무 번거로워. 한 곳에서 다 보고 싶다."
- **기대 가치:** 시간 절약, 중요 뉴스 놓치지 않기, 포트폴리오 통합 관리

#### 페르소나 B: 이수진 (28세, 스타트업 마케터)
- **직업:** 스타트업 마케터, 연봉 4,200만 원
- **투자 경력:** 1년차 (주식 입문)
- **투자 규모:** 주식 700만 원
- **행동 패턴:**
  - 투자에 관심이 생겼지만 어디서 공부해야 할지 모름
  - 유튜브로 재테크 콘텐츠를 자주 봄
  - 게임을 즐기며 성취감에 반응하는 성격
- **페인 포인트:** "투자 공부를 해야 하는데 뭘 봐야 할지도 모르겠고, 시작해도 금방 흥미를 잃어버려."
- **기대 가치:** 쉬운 투자 학습, 성취감·보상, 꾸준한 투자 습관 형성

#### 페르소나 C: 박성민 (41세, 중소기업 팀장)
- **직업:** 제조업 중소기업 팀장, 연봉 7,000만 원
- **투자 경력:** 10년차 (주식, 부동산)
- **투자 규모:** 주식 1.5억 원, 부동산 1채 보유
- **행동 패턴:**
  - 바쁜 일정 때문에 투자 정보 확인에 많은 시간을 쏟기 어려움
  - 핵심 정보만 빠르게 파악하길 원함
  - 규제 변화에 민감하며 부동산 시장 동향 주시
- **페인 포인트:** "중요한 부동산 규제 뉴스를 회의 중에 놓쳐서 대응을 못한 적이 있어. 핵심 알림만 받고 싶다."
- **기대 가치:** 규제/정책 알림, 시장 요약, 시간 효율성

## 3. 핵심 기능 요구사항

### 3.1 기능 우선순위 매트릭스

| 기능 | 중요도 | 구현 난이도 | 우선순위 |
|------|--------|------------|---------|
| 통합 대시보드 | 높음 | 중간 | P0 |
| 실시간 주식 시세 | 높음 | 중간 | P0 |
| 실시간 코인 시세 | 높음 | 낮음 | P0 |
| 규제/뉴스 알림 | 높음 | 중간 | P0 |
| 게이미피케이션 (XP/뱃지) | 높음 | 높음 | P0 |
| 일일 퀘스트 | 중간 | 중간 | P1 |
| 관심 종목 설정 | 높음 | 낮음 | P0 |
| 포트폴리오 입력/추적 | 중간 | 높음 | P1 |
| 부동산 정보 | 중간 | 높음 | P1 |
| 커뮤니티/소셜 | 낮음 | 높음 | P2 |
| AI 투자 인사이트 | 중간 | 높음 | P2 |

### 3.2 P0 — 핵심 기능 (MVP)

#### F-001: 통합 투자 대시보드
**설명:** 주식·코인·부동산 주요 지표를 한 화면에서 확인하는 메인 대시보드

**기능 상세:**
- 오늘의 시장 요약 (코스피, 코스닥, 비트코인, 환율 한눈에)
- 개인 관심 자산 빠른 조회 (최대 10개)
- 오늘의 주요 뉴스 헤드라인 3건
- 현재 접속 중인 유저 수 (커뮤니티 활성감 제공)
- 나의 오늘 XP 현황 및 퀘스트 달성률

**수용 기준:**
- 대시보드 로딩 2초 이내
- 시세 데이터 최신화 주기: 실시간(WebSocket) 또는 1분 간격 폴링
- 모바일(375px) ~ 데스크탑(1440px) 반응형 지원

#### F-002: 실시간 주식 시세
**설명:** 국내 주식(코스피/코스닥) 실시간 시세 조회

**기능 상세:**
- 종목 검색 (이름 또는 종목코드)
- 시세 정보: 현재가, 전일 대비 등락, 등락률, 거래량, 시가총액
- 간이 차트 (일/주/월/년 기간 선택)
- 관심 종목 추가/삭제
- 투자자별 매매 동향 (개인/외국인/기관)

**수용 기준:**
- 시세 지연 30초 이내 (한국투자증권 OpenAPI 기준)
- 차트 데이터 1년치 히스토리 제공
- 관심 종목 최대 50개 등록 가능

#### F-003: 실시간 코인 시세
**설명:** 국내외 암호화폐 실시간 시세 조회

**기능 상세:**
- 업비트 KRW 마켓 전 종목 조회
- 시세 정보: 현재가, 24시간 변동, 거래대금, 고가/저가
- 글로벌 시세 (바이낸스 BTC/USDT 기준)
- 김치 프리미엄 실시간 표시
- 관심 코인 설정

**수용 기준:**
- WebSocket 연결로 실시간 시세 반영 (1초 이내)
- 업비트 WebSocket API 활용
- 오류 발생 시 자동 재연결 (최대 3회)

#### F-004: 규제/정책 뉴스 스마트 알림
**설명:** 투자에 영향을 미치는 규제·정책 뉴스를 카테고리별로 분류하여 제공

**기능 상세:**
- 뉴스 카테고리: 금융규제, 부동산정책, 암호화폐규제, 기업공시, 기준금리
- 중요도 3단계 분류: 긴급(빨강)/주요(주황)/일반(흰색)
- 뉴스 출처: 금융위원회, 한국은행, 기획재정부, 주요 언론사
- 키워드 기반 알림 설정 (예: "금리", "삼성전자", "비트코인")
- 읽은 뉴스 표시, 북마크 기능

**수용 기준:**
- 중요 규제 뉴스 발행 후 5분 이내 노출
- 하루 뉴스 최소 20건 이상 제공
- 알림 설정 키워드 최대 20개

#### F-005: 게이미피케이션 — XP 및 레벨 시스템
**설명:** 투자 정보 소비 행동에 XP(경험치)를 부여하고 레벨을 성장시키는 시스템

**XP 획득 행동:**
| 행동 | XP |
|------|-----|
| 앱 출석 체크 (일 1회) | +10 XP |
| 뉴스 1건 읽기 | +5 XP |
| 규제 뉴스 읽기 | +15 XP |
| 관심 종목 시세 확인 | +3 XP |
| 일일 퀘스트 완료 | +50 XP |
| 주간 연속 출석 (7일) | +100 XP 보너스 |
| 경제 퀴즈 정답 | +20 XP |

**레벨 체계:**
| 레벨 | 명칭 | 필요 XP |
|------|------|---------|
| 1 | 투자 입문자 | 0 |
| 2 | 마켓 탐색자 | 500 |
| 3 | 정보 수집가 | 1,500 |
| 4 | 시장 관찰자 | 3,500 |
| 5 | 트렌드 분석가 | 7,000 |
| 6 | 섹터 전문가 | 12,000 |
| 7 | 마켓 인사이더 | 20,000 |
| 8 | 투자 마에스트로 | 35,000 |
| 9 | 월스트리트 레전드 | 60,000 |
| 10 | PulseM 마스터 | 100,000 |

**수용 기준:**
- XP 부여는 서버 사이드에서만 처리 (클라이언트 조작 방지)
- 동일 행동 중복 XP 방지 (일 1회 한도 행동 관리)
- XP 변동 시 실시간 애니메이션 피드백 (+10 XP 플로팅 텍스트)

#### F-006: 뱃지 및 업적 시스템
**설명:** 특정 행동 달성 시 뱃지를 수여하는 업적 시스템

**뱃지 목록 (초기 30개):**
- 첫 발걸음: 첫 로그인
- 7일의 약속: 7일 연속 출석
- 30일의 습관: 30일 연속 출석
- 뉴스 헌터: 뉴스 100건 읽기
- 규제 레이더: 규제 뉴스 50건 읽기
- 주식 탐험가: 주식 종목 50개 조회
- 코인 감지기: 코인 30개 조회
- 퀴즈왕: 경제 퀴즈 20회 연속 정답
- 부지런한 투자자: 오전 7시 이전 앱 접속 30회
- 야행성 분석가: 밤 11시 이후 앱 접속 30회

**수용 기준:**
- 뱃지 달성 시 전체 화면 축하 애니메이션 표시 (Framer Motion)
- 뱃지 획득 알림 푸시 발송
- 프로필 페이지에 획득한 뱃지 전시

### 3.3 P1 — 주요 기능

#### F-007: 일일 퀘스트 시스템
- 매일 오전 6시 갱신되는 3~5개 퀘스트
- 퀘스트 예시: "오늘 코스피 등락률 확인하기", "부동산 뉴스 2건 읽기", "오늘의 경제 퀴즈 풀기"
- 퀘스트 완료 시 XP 보너스 + 랜덤 아이템(코인 꾸미기 프레임 등)

#### F-008: 포트폴리오 추적
- 보유 종목/코인 수동 입력
- 매수 평균단가, 보유 수량 기록
- 현재 평가손익, 수익률 계산
- 자산 배분 파이 차트

#### F-009: 부동산 정보
- 아파트 실거래가 조회 (공공데이터포털 API)
- 지역별 부동산 규제 현황
- 청약 일정 캘린더
- 금리 및 부동산 관련 뉴스

#### F-010: 글로벌 시장 지수
- 미국 3대 지수 (다우, S&P 500, 나스닥)
- 주요국 환율 (USD/KRW, EUR/KRW, JPY/KRW)
- 원자재 (금, 원유)
- 공포탐욕지수 (Fear & Greed Index)

### 3.4 P2 — 추가 기능

#### F-011: AI 투자 인사이트
- 관심 종목 관련 뉴스 자동 요약

- 개인 포트폴리오 리스크 분석

#### F-012: 커뮤니티 (PulseM 토론방)
- 종목별 토론방
- 실명 인증 투자 현황 공개 (자발적)

#### F-013: 투자 캘린더
- FOMC 회의, 실적 발표 일정
- 국내 주요 경제 지표 발표 일정
- 개인 투자 일정 메모

## 4. 사용자 스토리

### 4.1 인증 및 온보딩

```
US-001: 소셜 로그인
As 신규 사용자
I want to 구글/카카오 계정으로 간편하게 가입하고 싶다
So that 별도 계정 생성 없이 바로 서비스를 이용할 수 있다

수용 기준:
- 구글, 카카오 OAuth 2.0 로그인 지원
- 첫 로그인 시 닉네임 설정 (2-20자, 한글/영문/숫자)
- 관심 자산 카테고리 선택 (주식/코인/부동산 중 복수 선택)
- 온보딩 완료 후 대시보드 이동 + 웰컴 XP 50 지급
```

```
US-002: 관심 분야 설정
As 온보딩 중인 사용자
I want to 내가 관심 있는 투자 분야와 관심 종목을 설정하고 싶다
So that 내 관심사에 맞춘 맞춤형 대시보드를 볼 수 있다

수용 기준:
- 관심 카테고리: 국내주식/미국주식/코인/부동산/ETF
- 관심 종목 최초 5개 이상 설정 유도
- 설정 후 즉시 대시보드에 반영
```

### 4.2 시세 조회

```
US-003: 오늘 아침 시장 확인
As 출근 중인 직장인 투자자
I want to 앱을 열자마자 오늘 시장 전반을 빠르게 파악하고 싶다
So that 5분 이내에 오늘 투자 현황을 파악하고 출근 준비를 할 수 있다

수용 기준:
- 코스피/코스닥/비트코인/달러 환율을 메인 화면 상단에 표시
- 등락에 따른 컬러 코딩 (상승: 빨강, 하락: 파랑 — 국내 관행)
- 화면 진입 시 데이터 자동 갱신
```

```
US-004: 관심 종목 상세 조회
As 특정 종목을 보유 중인 투자자
I want to 해당 종목의 오늘 주가 변동과 관련 뉴스를 한 화면에서 보고 싶다
So that 여러 앱을 오가지 않고 한 곳에서 의사결정에 필요한 정보를 얻을 수 있다

수용 기준:
- 종목 상세 페이지에 시세 차트 + 관련 뉴스 최근 5건 함께 표시
- 관련 뉴스는 종목명 기준 네이버/다음 뉴스 검색 결과 활용
- 관심 종목 추가 버튼 1탭으로 조작 가능
```

### 4.3 게이미피케이션

```
US-005: 일일 출석 체크
As 매일 앱을 여는 사용자
I want to 출석 체크를 하고 XP를 받고 싶다
So that 앱을 열 때마다 작은 보상감을 느끼고 꾸준히 사용하게 된다

수용 기준:
- 앱 첫 접속 시 자동으로 출석 처리 (별도 버튼 불필요)
- 출석 시 +10 XP 지급 + 애니메이션 피드백
- 연속 출석 일수 표시 (스트릭 카운터)
- 7일 연속 출석 달성 시 보너스 XP + 뱃지 획득
```

```
US-006: 퀘스트 완료
As 일일 퀘스트를 수행하는 사용자
I want to 오늘의 퀘스트를 완료하고 보상을 받고 싶다
So that 자연스럽게 다양한 투자 정보를 탐색하게 된다

수용 기준:
- 홈 화면 하단에 오늘의 퀘스트 3개 항상 표시
- 퀘스트 행동 수행 시 실시간 진행률 업데이트
- 퀘스트 완료 시 체크마크 애니메이션 + XP 획득 팝업
- 모든 퀘스트 완료 시 "오늘의 투자자" 칭호 일일 부여
```

```
US-007: 랭킹 확인
As 경쟁심이 있는 사용자
I want to 내가 다른 사용자들과 비교해 어느 위치인지 알고 싶다
So that 더 많은 정보를 탐색하고 XP를 쌓으려는 동기를 얻는다

수용 기준:
- 전체 랭킹, 주간 랭킹 탭 제공
- 내 순위가 항상 화면에 고정 표시 (상위 100명 목록 + 내 위치)
- 랭킹 상위 3명 특별 뱃지 시각화
- 랭킹 주간 초기화 시 이전 주 랭킹 보관
```

### 4.4 뉴스 및 알림

```
US-008: 중요 규제 뉴스 알림
As 규제 변화에 민감한 투자자
I want to 내 관심 자산과 관련된 중요 규제 뉴스가 나오면 즉시 알림을 받고 싶다
So that 시장 변화에 빠르게 대응할 수 있다

수용 기준:
- 설정한 키워드 포함 뉴스 발행 시 5분 이내 푸시 알림 발송
- 알림 클릭 시 해당 뉴스 상세 페이지로 딥링크 이동
- 긴급 뉴스(예: 기준금리 변경) 는 강제 알림으로 발송
- 알림 시간대 설정 가능 (방해금지 시간 설정)
```

## 7. 의존성 및 제약사항

### 7.1 외부 의존성

| 의존성 | 용도 | 리스크 | 대안 |
|--------|------|--------|------|
| 한국투자증권 OpenAPI | 주식 시세 | API 변경/중단 | 키움 OpenAPI |
| 업비트 API | 코인 시세 | 서비스 정책 변경 | 빗썸 API |
| 공공데이터포털 | 부동산 실거래가 | 데이터 갱신 지연 | 국토교통부 직접 |
| 네이버 뉴스 Search API | 뉴스 수집 | 호출 제한 | 다음 뉴스 API |
| Supabase | 백엔드/DB | 서비스 장애 | PlanetScale + Auth.js |

### 7.2 제약사항

- **한국투자증권 OpenAPI:** 실시간 데이터는 증권사 계좌 연동 필요. MVP에서는 지연 데이터(20분) 사용 후 점진적 개선
- **금융 정보 규제:** 투자 권유 또는 종목 추천 기능은 금융투자업 인가 필요. 정보 제공으로만 서비스 범위 한정
- **개인정보보호법:** 최소한의 개인정보만 수집, 회원 탈퇴 시 즉시 삭제 처리
