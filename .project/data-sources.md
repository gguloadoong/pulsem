# PulseM — Data Sources

## 1. 필요 데이터 소스
| 카테고리 | 데이터 | 갱신 주기 |
|---------|--------|----------|
| 주식 | KOSPI/KOSDAQ 종목 시세, 거래량, 차트 | 실시간 (장중) |
| 코인 | BTC/ETH/SOL 등 주요 코인 시세 | 실시간 (24/7) |
| 부동산 | 서울 구별 아파트 매매가, 거래량 | 주 1회 |
| 뉴스 | 투자 관련 뉴스, 규제/정책 변동 | 실시간 |
| 트렌딩 | 투자 관련 인기 검색어 | 10분 |

## 2. 후보 데이터 소스 비교
### 주식
| 소스 | 유형 | 비용 | 장점 | 단점 |
|------|------|------|------|------|
| 한국투자증권 Open API | REST/WS | 무료 | 실시간, 공식 | 계좌 필요 |
| KRX 정보데이터시스템 | REST | 무료 | 공식 데이터 | 지연 시세 |
| 네이버 금융 | 스크래핑 | 무료 | 풍부한 데이터 | 이용약관 주의 |

### 코인
| 소스 | 유형 | 비용 | 장점 | 단점 |
|------|------|------|------|------|
| 업비트 API | REST/WS | 무료 | 원화 마켓, 안정적 | Rate limit |
| 빗썸 API | REST | 무료 | 대안 | 기능 제한 |
| CoinGecko API | REST | 프리미엄 유료 | 글로벌 데이터 | 무료 제한적 |

### 부동산
| 소스 | 유형 | 비용 | 장점 | 단점 |
|------|------|------|------|------|
| 국토부 실거래가 API | REST | 무료 | 공식, 신뢰성 | 주 1회 갱신 |
| 한국부동산원 | REST | 무료 | 시세 지수 | API 제한 |

## 3. 데이터 수집 주기 및 캐싱
- 실시간 시세: WebSocket 구독, 클라이언트 메모리 캐시
- 뉴스: 5분 간격 폴링, Redis 캐시 (Phase 2)
- 부동산: 주 1회 배치 수집, PostgreSQL 저장
- 트렌딩: 10분 간격 집계

## 4. 법적·라이선스 고려사항
- 한국투자증권: 개인 비상업적 사용 가능, 상업적 사용 시 별도 계약
- 업비트: API 이용약관 준수, 과도한 요청 금지
- 공공데이터: 공공누리 제1유형 (출처 표시)
- 웹 스크래핑: robots.txt 준수, 서비스 이용약관 확인 필수

## 5. 데이터 품질 확보
- 다중 소스 교차 검증 (주식: KRX + 증권사 API)
- 이상치 탐지: 전일 대비 ±30% 이상 변동 시 검증
- 장애 대응: 소스 불가 시 최신 캐시 데이터 표시 + 타임스탬프 고지

## 6. 개인정보 보호
- 관심종목·검색 이력은 서버에 암호화 저장
- 로그인 없이 기본 데이터 조회 가능 (익명 모드)
- GDPR/개인정보보호법 준수: 데이터 삭제 요청 처리
