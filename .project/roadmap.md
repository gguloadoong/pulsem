# PulseM — Product Roadmap

## Phase 0: 프로젝트 셋업 ✅ (현재)
- [x] Next.js 15 프로젝트 초기화
- [x] 디자인 시스템 정의 (다크 테마, 글래스모피즘)
- [x] 핵심 UI 컴포넌트 구현 (대시보드, 카드, 네비게이션)
- [x] 게이미피케이션 시스템 UI (XP, 스트릭, 퀘스트, 배지)
- [x] GitHub 저장소 + CI/CD 구성
- [x] 프로젝트 문서 작성 (PRD, 디자인 스펙, 기술 스펙)

## Phase 1: MVP 개발 (핵심 기능 검증)
- [ ] Supabase 연동 (Auth, Database)
- [ ] 실시간 주식 시세 API 연동 (한국투자증권)
- [ ] 실시간 코인 시세 API 연동 (업비트)
- [ ] 부동산 데이터 API 연동 (공공데이터포털)
- [ ] 뉴스 피드 실시간 수집
- [ ] 사용자 인증 (이메일 + 카카오 소셜)
- [ ] 관심종목 CRUD
- [ ] 출석 체크인 서버 로직
- [ ] 퀘스트 시스템 백엔드
- [ ] PWA 설정 (오프라인, 앱 설치)

## Phase 2: 제품 완성 (디자인 고도화, 안정성 강화)
- [ ] 상세 차트 (캔들차트, 이동평균선)
- [ ] 푸시 알림 (고임팩트 뉴스, 가격 알림)
- [ ] 개인화 추천 (AI 기반 뉴스/종목 추천)
- [ ] 라이트 모드 지원
- [ ] Lucide React 아이콘 마이그레이션
- [ ] 성능 최적화 (코드 스플리팅, 이미지 최적화)
- [ ] E2E 테스트 자동화 (Playwright)
- [ ] 접근성 개선 (WCAG AA 완전 준수)

## Phase 3: 성장 & 수익화
- [ ] 광고 시스템 (네이티브 광고, 배너)
- [ ] 프리미엄 구독 (고급 분석, 알림 무제한)
- [ ] 커뮤니티 기능 (종목 토론, 투자 의견)
- [ ] 리더보드 (주간/월간 XP 랭킹)
- [ ] 소셜 기능 (친구 초대, 프로필 공유)
- [ ] 마케팅 랜딩페이지

## Phase 4: 장기 전략
- [ ] React Native 네이티브 앱
- [ ] 글로벌 시장 확장 (미국/일본 주식)
