# PulseM — 제품 로드맵

**버전:** 1.0.0
**최종 수정:** 2026-03-21

---

## 개요

PulseM의 로드맵은 4개 Phase로 구성된다. 각 Phase는 약 2개월을 기준으로 하며, 이전 Phase의 핵심 지표 달성을 확인한 후 다음 Phase로 진행한다.

| Phase | 기간 | 핵심 목표 | 성공 기준 |
|-------|------|---------|---------|
| Phase 1 — MVP | 2개월 | 핵심 가치 검증 | 3,000 MAU, 7일 리텐션 25% |
| Phase 2 — Growth | 2개월 | 기능 완성 & 성장 | 15,000 MAU, 30일 리텐션 20% |
| Phase 3 — Scale | 2개월 | 바이럴 & 수익화 준비 | 50,000 MAU, DAU/MAU 35% |
| Phase 4 — Monetize | 이후 | 지속 성장 & 수익 | 흑자 전환 |

---

## Phase 1 — MVP: 핵심 가치 검증

**기간:** 2026-04 ~ 2026-05 (2개월)
**목표:** "주식·코인 통합 시세 + 기본 게이미피케이션"으로 첫 사용자 유입 및 습관 형성 가능성 검증

### 1-1. 인프라 설정 (Week 1~2)

- [ ] Next.js 15 프로젝트 초기화 (pnpm, TypeScript strict)
- [ ] Tailwind CSS v4 + CSS 변수 기반 다크 테마 설정
- [ ] Supabase 프로젝트 생성 + 초기 스키마 마이그레이션
  - `profiles`, `user_stats`, `xp_transactions`, `watchlist` 테이블
  - RLS 정책 전체 적용
- [ ] Vercel 배포 연결 + GitHub Actions CI 파이프라인 구성
- [ ] ESLint + Prettier + Husky + lint-staged 설정
- [ ] `.env.example` 작성 + 모든 팀원 환경변수 공유

### 1-2. 인증 시스템 (Week 2~3)

- [ ] Supabase Auth 구글 소셜 로그인 구현
- [ ] Supabase Auth 카카오 소셜 로그인 구현
- [ ] 온보딩 플로우: 닉네임 설정 → 관심 자산 선택 → 웰컴 XP 50 지급
- [ ] 미들웨어 인증 체크 (보호된 라우트)
- [ ] 프로필 조회/수정 API

### 1-3. 시세 데이터 — 목 데이터 기반 UI (Week 2~4)

- [ ] 메인 대시보드 레이아웃 (헤더 + 그리드 + 바텀탭/사이드바)
- [ ] 4대 지수 히어로 카드 (코스피/코스닥/BTC/달러)
- [ ] 마켓 티커 배너 (무한 스크롤 애니메이션)
- [ ] 종목 카드 컴포넌트 (시세 + 등락 색상 + 틱 애니메이션)
- [ ] 시장 탭 페이지 (국내주식 / 코인 탭)
- [ ] 종목 검색 기능 (로컬 필터링)
- [ ] 관심 종목 추가/삭제 UI + Supabase 연동

### 1-4. 실제 API 연동 (Week 4~6)

- [ ] 업비트 REST API 연동 (코인 목록 + 시세)
- [ ] 업비트 WebSocket 실시간 시세 구독 클라이언트
- [ ] 한국투자증권 OpenAPI 연동 (주식 시세 30초 폴링)
- [ ] 글로벌 지수 (Alpha Vantage — S&P 500, 나스닥, 다우)
- [ ] USD/KRW 환율 (한국은행 ECOS API)
- [ ] 김치프리미엄 실시간 계산 (업비트 KRW ÷ 바이낸스 USDT × 환율)

### 1-5. 게이미피케이션 — 핵심 (Week 4~7)

- [ ] Supabase `award_xp()` Database Function 구현
- [ ] 일일 출석 체크인 API + XP 지급 (중복 방지)
- [ ] XP 진행바 + 레벨 배지 컴포넌트
- [ ] XP 획득 플로팅 텍스트 애니메이션 (+10 XP)
- [ ] Supabase Realtime XP 실시간 동기화
- [ ] 레벨업 애니메이션 (전체화면 연출)
- [ ] 연속 출석 스트릭 카운터 + 7일 보너스

### 1-6. 뉴스 피드 (Week 5~7)

- [ ] Vercel Cron 뉴스 수집 파이프라인 (네이버 뉴스 5분 폴링)
- [ ] 공공기관 RSS 수집 (금융위원회, 한국은행, 기재부)
- [ ] 뉴스 중요도 분류 로직 (긴급/주요/일반)
- [ ] 뉴스 피드 페이지 (카테고리 필터)
- [ ] 뉴스 읽기 시 XP 지급 (read_news 액션)
- [ ] 뉴스 북마크 기능

### 1-7. MVP 마무리 (Week 7~8)

- [ ] 스켈레톤 로딩 UI 전체 컴포넌트에 적용
- [ ] 빈 상태(empty state) UI 모든 목록에 적용
- [ ] 에러 핸들링 + Sonner 토스트 알림
- [ ] 모바일 반응형 최종 점검 (375px, 390px, 428px)
- [ ] Lighthouse 성능 측정 + LCP < 2.5초 달성
- [ ] 베타 테스터 20명 내부 테스트

**Phase 1 종료 기준:**
- 핵심 기능 동작: 시세 조회, 코인 WebSocket, 출석 체크인, XP 적립
- Lighthouse 성능 점수 > 85
- 베타 테스터 7일 리텐션 > 30%

## Phase 2 — Growth: 기능 완성 & 성장

**기간:** 2026-06 ~ 2026-07 (2개월)
**목표:** 핵심 기능을 완성하고 바이럴 요소를 추가하여 15,000 MAU 달성

### 2-1. 게이미피케이션 고도화

- [ ] 뱃지 시스템 30개 구현 (Database Trigger 기반 자동 달성 체크)
- [ ] 뱃지 획득 전체화면 축하 애니메이션 (Framer Motion)
- [ ] 일일 퀘스트 시스템 (매일 오전 6시 3개 갱신)
- [ ] 퀘스트 진행률 실시간 업데이트
- [ ] 전체/주간 랭킹 페이지 (Supabase Realtime)
- [ ] 프로필 페이지 뱃지 컬렉션 진열
- [ ] 주간 활동 히트맵 (GitHub 잔디 스타일)
- [ ] 경제 퀴즈 기능 (+20 XP 보상)

### 2-2. 포트폴리오 추적

- [ ] 포트폴리오 종목 수동 입력 (종목명, 수량, 평단가)
- [ ] 현재가 기준 평가손익 자동 계산
- [ ] 수익률 표시 (총 수익률, 개별 종목별)
- [ ] 자산 배분 파이 차트 (Recharts)
- [ ] 포트폴리오 수정/삭제

### 2-3. 부동산 정보

- [ ] 공공데이터포털 아파트 실거래가 API 연동
- [ ] 지역별 실거래가 목록 (시/구 드롭다운)
- [ ] 부동산 관련 뉴스 필터링
- [ ] 청약 일정 캘린더 (청약홈 RSS)
- [ ] 주요 부동산 규제 현황 요약 카드

### 2-4. 상세 차트

- [ ] Lightweight Charts 캔들차트 (일봉/주봉/월봉)
- [ ] 이동평균선 (5일, 20일, 60일)
- [ ] 거래량 바 차트
- [ ] 주식/코인 상세 페이지 (차트 + 관련 뉴스)

### 2-5. 알림 시스템

- [ ] Web Push Notification 구현 (Service Worker)
- [ ] 알림 설정 페이지 (키워드, 카테고리, 방해금지 시간)
- [ ] 긴급 뉴스 푸시 알림 (금리 변경, 서킷브레이커 등)
- [ ] 관심 종목 가격 알림 (목표가 설정)

### 2-6. 성능 & 품질

- [ ] Dynamic import 코드 스플리팅 (차트, 뱃지 축하 등)
- [ ] Vitest 핵심 유틸리티 테스트 작성 (포맷터, XP 계산)
- [ ] Playwright E2E 핵심 플로우 5개
- [ ] Lighthouse CI GitHub Actions 연동
- [ ] Sentry 에러 추적 설정

### 2-7. 랜딩 페이지

- [ ] 서비스 소개 랜딩 페이지 (`/`)
- [ ] 주요 기능 섹션 (스크린샷 + 설명)
- [ ] 앱 다운로드 대신 "무료 시작하기" CTA
- [ ] OG 이미지 + SEO 메타데이터

**Phase 2 종료 기준:**
- MAU 15,000 달성

- [ ] 프로필 공개 페이지 (`/profile/{nickname}`) — 공유 가능

- [ ] 관심 종목 관련 뉴스 자동 요약 (Claude API)

- [ ] 개인 포트폴리오 리스크 분석 보고서

### 3.4 글로벌 시장 확장

- [ ] 미국 개별 주식 조회 (AAPL, TSLA, NVDA 등)
- [ ] 나스닥/NYSE 인기 종목 시세
- [ ] 미국 주식 관련 영문 뉴스 + 한국어 요약 (Claude API)

### 3-5. 수익화 기반 준비

- [ ] 프리미엄 구독 플랜 설계 (기능 차별화 정의)
- [ ] Stripe 결제 연동 기술 검증
- [ ] 프리미엄 전용 기능 정의:
  - 관심 종목 무제한 (기본 50개 → 무제한)
  - 가격 알림 무제한 (기본 5개 → 무제한)
  - AI 인사이트 일 10회 (기본 1회)
  - 광고 없음
- [ ] 베타 프리미엄 유저 100명 모집 (얼리버드 할인)

### 3-6. 앱 성능 & 안정성

- [ ] Core Web Vitals 모든 지표 "Good" 범위 달성
- [ ] PWA 설정 (Service Worker, 앱 설치 가능)
- [ ] 오프라인 모드 (관심 종목 캐시 데이터 표시)
- [ ] WCAG 2.1 AA 접근성 감사 + 개선

**Phase 3 종료 기준:**
- MAU 50,000 달성
- DAU/MAU 비율 > 35%
- 프리미엄 관심 표명 유저 500명 이상

## Phase 4 — Monetize: 지속 성장 & 수익

**기간:** 2026-10 이후
**목표:** 지속 가능한 수익 모델 구축 + 모바일 앱 론칭

### 4-1. 프리미엄 구독 정식 출시

- [ ] 월 구독 (9,900원/월) + 연 구독 (79,000원/년)

- [ ] 구독 상태에 따른 기능 잠금/해제 미들웨어

- [ ] React Native + Expo 기반 iOS/Android 앱

### 4-4. 데이터 분석 강화

- [ ] 사용자 행동 분석 대시보드 (내부)

- [ ] 이탈 사용자 재유입 캠페인 (이메일)

### 4-5. 글로벌 확장 검토
